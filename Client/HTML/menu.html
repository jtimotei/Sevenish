Rules

The developer was too lazy to write these himself. The rules can be found here. Note that this version of the game variates from the normal one in the following way: "the player who led the first card may keep it alive for another round. To do this, the player must play another card of the same rank as the first card in the trick (or a seven)."

Development

The game was implemented in the memory of my time in highschool, where my colleagues and I were playing this game almost every break. Since we went to different universities, we haven't had the possibility to play the game so I decided to create this website.
